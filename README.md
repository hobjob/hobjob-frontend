### HobJob (frontend) - Платформа для прохождения и покупки курсов

## Стек технологий

-   ReactJs + Hook
-   react-router-dom
-   Redux
-   Redux thunk
-   node-sass
-   axios

## Установка

Клонируйте репозиторий.

```
git clone https://github.com/hobjob/hobjob-frontend.git
```

Переход в рабочий каталог.

```
cd hobjob-frontend
```

Установка зависимостей, запуск сборки.

```
npm install
npm start
```
